## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation NestJS

```bash
$ npm install
```

## Installation Swagger

```bash
$ npm install --save @nestjs/swagger
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
